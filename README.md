# Transport-management-system
A python app for tracking daily transfers and calculating bonuses.
🚚 Transportation Management System
This application is for transportation companies or professional transporters who collaborate with transportation companies.
What you can do with the application:
Register your routes daily
Calculate the average transportation
Detailed view of transportation registration
Reset data for a new month
Easy progress tracking for bonus calculation
